### Manual Task 2: RLS Audit & Fixes ✅
Ran RLS audit queries. All 31 public tables had `rowsecurity = true` — no tables were completely unprotected.

**Fixed:**

1. **Broken member workflows** — `inventory_transactions`, `tank_inventory`, `tank_fills`, `tank_movements` write policies were admin-only, blocking regular members from pack/receive/unpack operations. Changed to allow all accepted org members.

2. **Self-join security hole** — `organization_members` INSERT policy allowed any authenticated user to add themselves to any org just by knowing the org ID. Tightened to require either being the org creator or having a pending invite for their email.

**SQL run in Supabase SQL Editor:**
```sql
-- inventory_transactions
DROP POLICY "Org admins can insert inventory transactions" ON inventory_transactions;
CREATE POLICY "Org members can insert inventory transactions"
ON inventory_transactions FOR INSERT
WITH CHECK (organization_id IN (
  SELECT organization_id FROM organization_members
  WHERE user_id = auth.uid() AND accepted = true
));

-- tank_inventory
DROP POLICY "Org admins can insert tank inventory" ON tank_inventory;
DROP POLICY "Org admins can update tank inventory" ON tank_inventory;
DROP POLICY "Org admins can delete tank inventory" ON tank_inventory;
CREATE POLICY "Org members can insert tank inventory"
ON tank_inventory FOR INSERT
WITH CHECK (organization_id IN (
  SELECT organization_id FROM organization_members
  WHERE user_id = auth.uid() AND accepted = true
));
CREATE POLICY "Org members can update tank inventory"
ON tank_inventory FOR UPDATE
USING (organization_id IN (
  SELECT organization_id FROM organization_members
  WHERE user_id = auth.uid() AND accepted = true
));
CREATE POLICY "Org members can delete tank inventory"
ON tank_inventory FOR DELETE
USING (organization_id IN (
  SELECT organization_id FROM organization_members
  WHERE user_id = auth.uid() AND accepted = true
));

-- tank_fills
DROP POLICY "Org admins can insert tank fills" ON tank_fills;
CREATE POLICY "Org members can insert tank fills"
ON tank_fills FOR INSERT
WITH CHECK (organization_id IN (
  SELECT organization_id FROM organization_members
  WHERE user_id = auth.uid() AND accepted = true
));

-- tank_movements
DROP POLICY "Org admins can insert tank movements" ON tank_movements;
CREATE POLICY "Org members can insert tank movements"
ON tank_movements FOR INSERT
WITH CHECK (organization_id IN (
  SELECT organization_id FROM organization_members
  WHERE user_id = auth.uid() AND accepted = true
));

-- organization_members self-join fix
DROP POLICY "Users can insert own membership" ON organization_members;
CREATE POLICY "Users can insert own membership"
ON organization_members FOR INSERT
WITH CHECK (
  auth.uid() = user_id
  AND (
    EXISTS (
      SELECT 1 FROM organizations
      WHERE id = organization_id AND created_by = auth.uid()
    )
    OR
    EXISTS (
      SELECT 1 FROM pending_invites
      WHERE organization_id = organization_members.organization_id
      AND invited_email = (auth.jwt() ->> 'email')
      AND accepted = false
    )
  )
);
```
